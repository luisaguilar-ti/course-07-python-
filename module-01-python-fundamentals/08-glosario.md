# 08 - Glosario

## Introducción

Este glosario reúne los principales términos presentados durante el Módulo 1 del curso **Automatizar tareas de ciberseguridad con Python**. Su propósito es servir como material de consulta rápida para reforzar los conceptos fundamentales del lenguaje Python y la automatización.

---

# Glosario

## A

### Automatización
Uso de la tecnología para reducir el esfuerzo humano y manual en la realización de tareas comunes y repetitivas.

---

## C

### Comentario
Nota que hacen los programadores sobre la intención detrás de su código.

### Datos booleanos (Boolean)
Datos que solo pueden tener uno de dos valores: `True` o `False`.

### Datos de cadena (String)
Datos que consisten en una secuencia ordenada de caracteres.

### Datos de conjunto (Set)
Datos que consisten en una colección desordenada de valores únicos.

### Datos de diccionario (Dictionary)
Datos que constan de una o varias asociaciones clave-valor.

### Datos de lista (List)
Estructura de datos que consiste en una colección ordenada de elementos.

### Datos de tupla (Tuple)
Estructura de datos que consiste en una colección de elementos que no pueden modificarse después de su creación.

### Datos enteros (Integer)
Datos que representan un número sin punto decimal.

### Datos flotantes (Float)
Datos que representan un número con punto decimal.

---

## E

### Entorno de desarrollo integrado (IDE)
Aplicación de software para escribir código que proporciona herramientas de edición, depuración y asistencia al desarrollo.

### Error de tipo (TypeError)
Error que ocurre al utilizar un tipo de dato incorrecto en una operación.

---

## I

### Interfaz de línea de comandos (CLI)
Interfaz basada en texto que utiliza comandos para interactuar con una computadora.

### Intérprete
Programa que traduce el código Python en instrucciones ejecutables línea por línea.

---

## N

### Notebook
Interfaz interactiva utilizada para escribir, almacenar y ejecutar código.

---

## P

### Programación
Proceso utilizado para crear un conjunto de instrucciones que una computadora ejecuta para realizar tareas.

---

## S

### Sentencia condicional
Sentencia que evalúa una condición para decidir qué instrucciones ejecutar.

### Sentencia iterativa
Código que ejecuta repetidamente un conjunto de instrucciones.

### Sintaxis
Conjunto de reglas que determina cómo debe escribirse correctamente un lenguaje de programación.

---

## T

### Tipo de dato
Categoría que clasifica un tipo específico de información.

---

## V

### Variable
Contenedor utilizado para almacenar datos.

### Variable de bucle
Variable utilizada para controlar las iteraciones de un bucle.

---
